# 🌵 Olá, Peregrino!

Bem-vindo ao oásis digital do Paxá! Aqui no blog, navegamos pelas vastas
dunas da tecnologia, explorando o império do desenvolvimento web,
programação e administração de sistemas com uma pitada de cultura geek.

Este espaço é uma extensão do meu trabalho como Desenvolvedor Web, onde
compartilho artigos, dicas, snippets de código e alguns bookmarks. Se você é
um nômade digital em busca de conhecimento, está no lugar certo!

No blog, você encontrará artigos e snippets sobre Astro, Vue.js, Vite, Arch Linux, Rocky Linux, Podman e etc. Aqui estão algumas
ideias para começar sua jornada:    

- [Documentação da Sintaxe MDX](https://mdxjs.com/docs/what-is-mdx)
- [Documentação de uso do Astro](https://docs.astro.build/en/guides/markdown-content/#markdown-and-mdx-pages)
- **Nota:** [Diretivas lado cliente](https://docs.astro.build/en/reference/directives-reference/#client-directives) são obrigatórias para criar componentes interativos. De outro modo, todos os componentes no seu MDX irão renderizar como HTML estático (sem JavaScript) por padrão.

Aproveite a viagem! Se precisar de ajuda, lembre-se de comentar as postagens
ou <a href="mailto:paxa@paxa.dev">me enviar um e-mail</a> para fazer perguntas.
